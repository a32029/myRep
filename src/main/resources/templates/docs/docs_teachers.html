Teachers

This document was designed to help a client to access a management system via an HTTP API.

  • Create Teacher
    • REST method:
      • POST
    • Parameters:
      • JSON Object { "number":teacher_id, "name":teacher_name, "email":teacher_email, "role":( choose "ROLE_USER" or "ROLE_ADMIN") }
    • Result:
      • The Teacher id.

  • Update Teacher
    • REST method:
      • POST
    • Parameters:
      • JSON Object { "number":teacher_id, "name":teacher_name, "email":teacher_email, "role":( choose "ROLE_USER" or "ROLE_ADMIN") }
    • Result:
      • The Teacher id.

  • Get Teacher
    • REST method:
      • GET
    • Parameters:
      • teacher_id={teacher_id}
    • Result:
      • JSON Object { "number":teacher_id, "name":teacher_name, "email":teacher_email, "role":( "ROLE_USER" or "ROLE_ADMIN") }

  • List Teachers
    • REST method:
      • GET
    • Parameters:
      • p={page number}
      • l={number of rows per page}
      • sp={search parameter}
      • sp={search value (the character * is a wildcard)}
    • Result:
      • JSON Object[] [{ "number":teacher_id, "name":teacher_name, "email":teacher_email, "role":( "ROLE_USER" or "ROLE_ADMIN") }]
