Students

This document was designed to help a client to access a management system via an HTTP API.

  • Create Student
    • REST method:
      • POST
    • Parameters:
      • JSON Object { "number":student_id, "name":student_name, "email":student_email, "role":( choose "ROLE_USER" or "ROLE_ADMIN") }
    • Result:
      • The Student id.

  • Update Student
    • REST method:
      • POST
    • Parameters:
      • JSON Object { "number":student_id, "name":student_name, "email":student_email, "role":"ROLE_USER" }
    • Result:
      • The Student id.

  • Get Student
    • REST method:
      • GET
    • Parameters:
      • student_id={student_id}
    • Result:
      • JSON Object { "number":student_id, "name":student_name, "email":student_email, "role":"ROLE_USER" }

  • List Students
    • REST method:
      • GET
    • Parameters:
      • p={page number}
      • l={number of rows per page}
      • sp={search parameter}
      • sp={search value (the character * is a wildcard)}
    • Result:
      • JSON Object[] [{ "number":student_id, "name":student_name, "email":student_email, "role":"ROLE_USER" }]
