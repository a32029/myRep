Group in Class

This document was designed to help a client to access a management system via an HTTP API.

  • Join Group in Class
    • REST method:
      • PUT
    • Parameters:
      • class_id={class_id}
      • group_id={group_id}
      • student_id={student_id}
    • Result:
      • none

  • Leave Group in Class
    • REST method:
      • DELETE
    • Parameters:
      • class_id={class_id}
      • group_id={group_id}
      • student_id={student_id}
    • Result:
      • none

  • Get Student in Class
    • REST method:
      • GET
    • Parameters:
      • class_id={class_id}
      • group_id={group_id}
      • student_id={student_id}
    • Result:
      • JSON Object { "class_id":class_id, "group_id":group_id, "student_id":student_id, "name":name, "email":email, "role":"ROLE_USER" }

  • List Group in Class
    • REST method:
      • GET
    • Parameters:
      • class_id={class_id}
      • p={page number}
      • l={number of rows per page}
      • sp={search parameter}
      • sv={search value (the character * is a wildcard)}
    • Result:
      • JSON Object[] [{ "class_id":class_id, "group_id":group_id, "student_id":student_id, "name":name, "email":email, "role":"ROLE_USER" }]
