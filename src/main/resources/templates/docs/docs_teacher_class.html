Teachers in Class

This document was designed to help a client to access a management system via an HTTP API.

  • Add Teacher to Class
    • REST method:
      • PUT
    • Parameters:
      • class_id={class_id}
      • teacher_id={teacher_id}
    • Result:
      • none

  • Remove a Teacher from Class
    • REST method:
      • DELETE
    • Parameters:
      • class_id={class_id}
      • teacher_id={teacher_id}
    • Result:
      • none

  • Get Teacher in Class
    • REST method:
      • GET
    • Parameters:
      • class_id={class_id}
      • teacher_id={teacher_id}
    • Result:
      • JSON Object { "number":teacher_id, "name":teacher_name, "email":teacher_email, "role":( "ROLE_USER" or "ROLE_ADMIN") }

  • List Teachers in Class
    • REST method:
      • GET
    • Parameters:
      • class_id={class_id}
      • p={page number}
      • l={number of rows per page}
    • Result:
      • JSON Object[] [{ "number":teacher_id, "name":teacher_name, "email":teacher_email, "role":( "ROLE_USER" or "ROLE_ADMIN") }]
