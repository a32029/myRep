Students in Class

This document was designed to help a client to access a management system via an HTTP API.

  • Add Student to Class
    • REST method:
      • PUT
    • Parameters:
      • class_id={class_id}
      • student_id={student_id}
    • Result:
      • none

  • Remove a Student from Class
    • REST method:
      • DELETE
    • Parameters:
      • class_id={class_id}
      • student_id={student_id}
    • Result:
      • none

  • Get Student in Class
    • REST method:
      • GET
    • Parameters:
      • class_id={class_id}
      • student_id={student_id}
    • Result:
      • JSON Object { "number":student_id, "name":student_name, "email":student_email, "role":( "ROLE_USER" or "ROLE_ADMIN") }

  • List Students in Class
    • REST method:
      • GET
    • Parameters:
      • class_id={class_id}
      • p={page number}
      • l={number of rows per page}
    • Result:
      • JSON Object[] [{ "number":student_id, "name":student_name, "email":student_email, "role":( "ROLE_USER" or "ROLE_ADMIN") }]
