Semesters

This document was designed to help a client to access a management system via an HTTP API.

  • Create Semester
    • REST method:
      • POST
    • Parameters:
      • JSON Object { "start_year":start_year, "end_year":end_year, "season":( choose "winter" or "summer") }
    • Result:
      • The Semester id.

  • Remove Semester
    • REST method:
      • DELETE
    • Parameters:
      • semester_id={semester_id}
    • Result:
      • The Semester id.

  • Get Semester
    • REST method:
      • GET
    • Parameters:
      • semester_id={semester_id}
    • Result:
      • JSON Object { "start_year":start_year, "end_year":end_year, "season":( choose "winter" or "summer") }

  • List Semesters
    • REST method:
      • GET
    • Parameters:
      • p={page number}
      • l={number of rows per page}
      • sp={search parameter}
      • sp={search value (the character * is a wildcard)}
    • Result:
      • JSON Object[] [{ "start_year":start_year, "end_year":end_year, "season":( choose "winter" or "summer") }]
