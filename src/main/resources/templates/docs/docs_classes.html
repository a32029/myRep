Classes

This document was designed to help a client to access a management system via an HTTP API.

  • Create Class
    • REST method:
      • POST
    • Parameters:
      • JSON Object { "class_id":class_id, "identifier":identifier, "course_id":course_id, "semester_id":semester_id, "auto_enrollment":true/false (permits students to join a class), "max_students_group":(maximum number of students per group) }
    • Result:
      • The Class id.

  • Update Class
    • REST method:
      • POST
    • Parameters:
      • JSON Object { "class_id":class_id, "identifier":identifier, "course_id":course_id, "semester_id":semester_id, "auto_enrollment":true/false (permits students to join a class), "max_students_group":(maximum number of students per group) }
    • Result:
      • The Class id.

  • Get Class
    • REST method:
      • GET
    • Parameters:
      • class_id={class_id}
    • Result:
      • JSON Object { "class_id":class_id, "identifier":identifier, "course_id":course_id, "semester_id":semester_id, "auto_enrollment":true/false (permits students to join a class), "max_students_group":(maximum number of students per group) }

  • List Classes
    • REST method:
      • GET
    • Parameters:
      • p={page number}
      • l={number of rows per page}
      • sp={search parameter}
      • sp={search value (the character * is a wildcard)}
    • Result:
      • JSON Object[] [{ "class_id":class_id, "identifier":identifier, "course_id":course_id, "semester_id":semester_id, "auto_enrollment":true/false (permits students to join a class), "max_students_group":(maximum number of students per group) }]
